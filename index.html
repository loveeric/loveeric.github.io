什么是GitHub页面？

准备开始了吗？卷起香草，或为您的项目生成一个网站。

您可以通过GitHub帐户和组织
以及无限的项目网站获得一个网站。准备？让我们开始吧。

  • 用户或组织站点
  • 项目现场

  • 创建一个存储库

    转到GitHub，并创建一个名为username .github.io 的新存储库，其中username是您在GitHub上的用户名（或组织名称）。

    如果存储库的第一部分与您的用户名不完全匹配，它将无法正常工作，因此请确保正确。

  • 克隆资源库

    转到要存储项目的文件夹，然后克隆新存储库：

    〜$ git clone https://github.com/ username / username .github.io

  • Clone the repository

    Click the "Set up in Desktop" button. When the GitHub desktop app opens, save the project.

    If the app doesn't open, launch it and clone the repository from the app.

  • 你好，世界

    输入项目文件夹并添加一个index.html文件：

    〜$ cd username .github.io

    〜$ echo“Hello World”> index.html

  • Create an index file

    Grab your favorite text editor and add an index.html file to your project:

    index.html
    <!DOCTYPE html>
<html>
<body>
<h1>Hello World</h1>
<p>I'm hosted with GitHub Pages.</p>
</body>
</html>

  • 推它

    添加，提交和推送更改：

    〜$ git add --all

    〜$ git commit -m“初始提交”

    〜$ git push -u原始主

  • 提交和同步

    输入存储库，提交更改，然后按同步按钮。

  • 提交和同步

    输入存储库，提交更改，然后按同步按钮。

  • ...你完成了！

    启动浏览器并转到https：// username .github.io。

  • 使用主题，或从头开始？

    您可以选择从其中一个预构建的主题开始，
    或者从头开始创建一个站点。

    • 选择一个主题
    • 白手起家

  • 存储库设置

    转到GitHub.com并创建一个新的存储库，或转到现有存储库。
    单击设置选项卡。

  • 主题选择

    向下滚动到GitHub Pages部分。按选择主题。

  • 选一个主题

    选择顶部旋转木马的主题之一。
    完成后，点击右侧的选择主题。

  • 编辑内容

    使用编辑器将内容添加到您的网站。

  • 承诺

    输入提交评论，然后点击编辑器下方的提交更改。

  • 创建一个索引文件

    转到GitHub.com并创建一个新的存储库，或转到现有存储库。
    单击创建新文件按钮。

  • 你好，世界

    命名文件index.html并在编辑器中输入一些HTML内容。

  • 提交文件

    滚动到页面的底部，写一个提交消息，并提交新的文件。

  • 存储库设置

    单击设置选项卡，向下滚动到GitHub页面部分。
    然后选择主分支源，然后单击保存按钮。

  • ...你完成了！

    启动浏览器，然后访问http：// username .github.io / repository。
